# 8051 LED Interfacing

## Description
This project demonstrates LED interfacing using the AT89C51 microcontroller programmed in Embedded C.

## Components Used
- AT89C51 Microcontroller
- LED

## Software Used
- Keil uVision
- Proteus 8

## Files
- LED.c – Embedded C source code
- LED.uvproj – Keil project file
- STARTUP.A51 – Startup file
- Circuit.png – Proteus circuit diagram
